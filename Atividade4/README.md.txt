## 📂 Estrutura da Pasta

A pasta **Atividade 4** contém **2 arquivos Java**:

1. **Automovel.java**

   * Classe que representa um automóvel com os seguintes atributos:

     * `cor` (String)
     * `marca` (String)
     * `tamanho` (int)
     * `velocidade` (int, privado)
   * Métodos principais:

     * `acelerar(int aceleracao)` → aumenta a velocidade.
     * `inicializaVelocidade(int inicial)` → define um valor inicial para a velocidade.
     * `retornaVelocidade()` → retorna a velocidade atual do automóvel.

2. **TestaAutomovel.java**

   * Classe principal com o método `main`.
   * Cria dois automóveis com valores iniciais diferentes.
   * Exibe no console a velocidade atual de cada automóvel.

---

## 🎯 Objetivo da Atividade

* Praticar **construtores e encapsulamento** em POO.
* Utilizar atributos públicos e privados para reforçar boas práticas.
* Implementar métodos que simulam o comportamento de um automóvel (aceleração e velocidade).

---

## 💻 Tecnologias Utilizadas

* **Linguagem:** Java
* **Versão mínima:** Java 8+
* **IDE recomendada:** IntelliJ IDEA / Eclipse / VS Code

---

## 🚀 Execução

Ao executar o programa:

* Dois automóveis são criados (`Fiat` e `Honda`), cada um com cor, tamanho e velocidade inicial.
* O programa imprime no console a velocidade inicial de cada automóvel.
